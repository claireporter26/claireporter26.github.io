Coding Bootcamp Testimonials Slider

Design by Frontend Mentor. Code written by Claire Porter.

See it Live!

Using the design and specifications supplied by the FrontEnd Mentor website I experimented with different positioning methods for the multiple images that made up the background of this one page site, and I also created my first JavaScript slider.

Languages Used

  • HTML5
  • CSS
  • SCSS
  • FlexBox
  • Javascript

What I learned

This was the second project I have attempted from the Frontend Mentor website and it allowed me to practice using the techniques I had already learnt about to create a pre-set design. On this project I practiced using the HTML and CSS skills I had already acquired and then built on this knowledge, but the main skill I was excited to learn on this project was how you can create a slider in Javascript. Being a new language for me I had not previously put the knowledge I had acquired into practice on a real project, I was amazed to discover there were many different approaches I could have taken and this has left me excited to learn more.
