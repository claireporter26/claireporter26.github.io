Photograph Filter Project

See it Live!

This one page website allows users to filter a selection of photographs by type. I used CSS Grid to display the photographs for larger screens and JavaScript to add the functionality to the buttons.

Languages Used

  • HTML5
  • CSS
  • SCSS
  • CSS Grid
  • JavaScript

What I learned

This was a fun project to create. My main aim was to practice using JavaScript to add the functionality to the filter buttons so I didn't focus too heavily on design, however I feel the minimalist theme works well to showcase the photographs.
